是时候建立大一统的mm管理了！


虚拟地址（32位）结构：
┌───────────────┬───────────────┬───────────────────┐
│ 页目录索引(10) │ 页表索引(10)   │ 页内偏移(12)      │
│ 高10位         │ 中间10位      │ 低12位            │
├───────────────┼───────────────┼───────────────────┤
│       0x004   │      0x001    │        0x234      │ （示例值）
└───────────────┴───────────────┴───────────────────┘
        ↑           ↑               ↑
        │           │               │
        ├───────────┼───────────┼───▼───────────┤
        │           │               物理页内位置
        │           └─── 页表索引 → 查找页表项（存储物理页框号）
        └─── 页目录索引 → 查找页目录项（指向对应页表）


物理地址转换过程：
┌───────────────┬───────────────────┐
│ 物理页框号(20) │ 页内偏移(12)       │
│ (页表项高20位）│ （直接来自虚拟地址）│
├───────────────┼───────────────────┤
│      0x00123  │        0x234      │ （示例值）
└───────────────┴───────────────────┘
        ↑               ↑
        │               └─── 直接拼接（低12位不变）
        └─── 左移12位 → 0x00123 << 12 = 0x00123000（物理页基地址）


最终物理地址：
───────────────────────────────▼───────────────
                          0x00123000 + 0x234 = 0x00123234
